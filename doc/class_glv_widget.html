Detailed Description

template<class Tvalue>
class GlvWidget< Tvalue >

Front-end of automatic widget management. GlvWidget -> GlvWidgetMaker -> GlvWidgetData. No text information here (unlike GlvDescribedWidget/GlvParameterWidget).
