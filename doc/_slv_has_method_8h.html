Macro for method detection in a struct/class/type, for instance when using std::enable_if. struct_name is the name of the structure to be used such as: std::enable_if<struct_name<T>::value> method_name is the method struct_name is supposed to contain such as: struct_name::method_name. method_name must be a public method.
