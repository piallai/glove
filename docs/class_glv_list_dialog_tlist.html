Detailed Description

template<class Tlist>
class GlvListDialogTlist< Tlist >

QDialog list of Tlist type. Tlist must have:

  • static const std::vector<std::string>& list() method
  • static const std::string& name() method No GlvParametrizationDialog management as compared to GlvParamListDialog.
