Detailed Description

template<class Tvalue, class Tlabel>
class SlvPoolFactory< Tvalue, Tlabel >

Same as SlvPool, but get method creates the element if not found. This auto get function only work if Tvalue has a constructor taking only Tlabel as argument. Otherwise, use SlvPool.
