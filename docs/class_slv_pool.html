Detailed Description

template<class Tvalue, class Tlabel = typename Tvalue::Tlabeling>
class SlvPool< Tvalue, Tlabel >

This class is a sort of std::map, with "factory" features. Tvalue must inherit SlvLabeling<Tlabel>, with Tlabel being the 'key'. Therefore, unlike std::map, the key (Tlabel) is owned by Tvalue. The instance owns the elements (no shared pointer).
