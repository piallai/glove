template<class Tparam>
class SlvParameterRuleT< SlvParameter< Tparam >, typename std::enable_if<(!slv::ts::is_arithmetic< Tparam >::value||std::is_same< Tparam, bool >::value) >::type >

Rule that a parameter SlvParameter is supposed to abide. Same as SlvParameterRuleT_spec_Arithmetic, but instead of using static values for rule, uses a dynamic value in a SlvParameter. See dynamic_rules in SlvParameter.
