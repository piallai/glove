Detailed Description

template<class Tlist, class Toption>
class GlvParamListDialog< Tlist, Toption >

Configuration list of Tlist type. Tlist must have:

  • static const std::vector<std::string>& list() method
  • static const std::string& name() method Toption is the type of the optional data provided for the parametrization windows. See GlvParamListDialog_Open::open. The optional data is not stored in the GlvParamListDialog instance.
