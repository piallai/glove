Detailed Description

template<class Tparametrization>
class GlvParametrizationWidget< Tparametrization >

Widget managing SlvParametrization to widget, and vice versa. Tparametrization is a class inheriting from SlvParametrization. Member values not belonging to SlvParametrization are ignored here (constructor values). This is the main widget (with GlvParametrizationDialog) to use for parametrization management in the framework.
