Detailed Description

template<class Tenum>
class SlvEnum< Tenum >

Class in charge of enum management. Not recommended to use directly. Use glvm_SlvEnum of glvm_SlvEnum_simple for instantiation. Example: glvm_SlvEnum(EnumName, A, B, C, D), or glvm_SlvEnum_named(EnumName, A, "A", B, "B", C, "C", D, "D").
