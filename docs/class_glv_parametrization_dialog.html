Detailed Description

template<class Tparametrization>
class GlvParametrizationDialog< Tparametrization >

Dialog widget managing parametrization widget. Tparametrization is a class inheriting from SlvParametrization. Member values not belonging to SlvParametrization are ignored here (constructor values). This is the main widget (with GlvParametrizationWidget) to use for parametrization management in the framework. Accept : set the parametrization has the one configured by the widget. Reject : does not modify the parametrization.
