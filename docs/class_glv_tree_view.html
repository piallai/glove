Detailed Description

template<class Tdata, template< class _Tdata > class Twidget_final, bool IsPersistent>
class GlvTreeView< Tdata, Twidget_final, IsPersistent >

Makes a table view from a template data. Caution: work in progress for proper management of delegates size. If a glv::tdata::QStandardItemModelMaker specialization exists, each delegate item is a GlvTreeView. Can manage delegate's editor space by adding temporary header sections. Edition mode is forced to double click if delegate is modeled. It prevents clicking on temporary sections.
