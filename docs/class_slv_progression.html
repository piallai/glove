Detailed Description

Use Qt signals.

Convenience class to embed a SlvProgressionQt in a class. Typical use: to be inherited by a class which contains a method to monitor. The key class is actually SlvProgressionQt, but it is convenient to encapsulate it in SlvProgression in order to not add too many inherited methods in inheriting classes. Also convenient to track the get_progression() method.
