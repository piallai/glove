Detailed Description

Class is in charge to bridge signals from GlvWidgetData to GlvWidget_base. Basically, any GlvWidget<T> can emit ValueChanged() for any basic type The connector is here to bridge signals of particular types to slots of this class. Ex: void valueChanged_slot(double _value). Each time valueChanged_slot is called by connectionn it emits the corresponding (type wise) signal valueChanged(xxx), and emits glvwidget's valueChanged() signal. Work in progress to add other types.
